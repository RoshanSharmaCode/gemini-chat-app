# Gemini Clone - AI Chatbot

A functional clone of Google Gemini built using React.js and the Google Gemini API. This project demonstrates how Generative AI can be integrated into a modern web application to provide real-time conversational experiences.

## 🚀 Live Features

- 🤖 AI-powered responses using Google Gemini API
- 💬 Real-time conversational chatbot
- 📜 Recent chat history management
- 🎯 Context API for global state management
- ⚡ Typing animation effect for AI responses
- 🔄 Loading animation while generating responses
- 📝 Automatic formatting for bold text and line breaks
- 📱 Fully responsive design for desktop and mobile
- 🆕 New Chat functionality
- 📂 Collapsible sidebar navigation

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/gemini-clone-react.git
```

### Navigate to Project

```bash
cd gemini-clone-react
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the root directory:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Application

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:5173
```

## 🔑 Getting Gemini API Key

1. Visit Google AI Studio
2. Create an API Key
3. Add the key to your `.env` file
